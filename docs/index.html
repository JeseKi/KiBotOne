ROS 2 · Gazebo Sim · 模式控制 · 自主避障

KiBotOne

一个面向移动机器人控制实验的 ROS 2 仿真项目，覆盖手动控制、巡航、停止、目标跟随与基于激光雷达的局部避障导航。

查看手操演示 查看避障导航 视频演示

项目成果

当前系统已经形成了可启动、可切换模式、可手操、可跟随和可避障的完整仿真闭环。

4 种运行模式 STOP、CRUISE、MANUAL、FOLLOW 四种运行模式。
6 种位姿数据 桥接速度、里程计、TF、雷达、机器人位姿和目标位姿。
1 个统一入口 统一 bringup 入口，直接拉起仿真、桥接和核心控制节点。

核心模块

页面中的演示是前端 HTML 复现，实际项目实现位于 ROS 2 包中。

kibot_one_sim

Gazebo 仿真资源

提供机器人模型、跟随旗标、普通世界和障碍物世界，并通过 launch 自动设置模型资源路径。

kibot_one_control

模式与速度控制

模式服务负责 STOP、CRUISE、MANUAL、FOLLOW 切换，速度看门狗负责超时停车。

follow_controller

局部避障跟随

控制器结合机器人位姿、目标位姿和 LaserScan，在候选航向中选择更安全的运动方向。

手操 HTML 演示

直接使用键盘 WASD 或方向键控制机器人，按住时持续发布速度，松开后立即停止；Enter 或 Space 也可停车。

MANUAL 控制

机器人会持续显示最近轨迹，并模拟 `/cmd_vel_raw` topic 的按住发布、松开停止行为。

  • 线速度 0.00 m/s
  • 角速度 0.00 rad/s
  • 最近命令 停止

自主避障导航 HTML 演示

演示会模拟机器人朝目标点移动，并在障碍物附近选择更安全的航向。可直接拖动橙色目标点和灰色障碍物。

FOLLOW + 避障

蓝色轨迹表示机器人走过的路径，橙色目标点对应跟随旗标，灰色圆形表示障碍物，外侧半透明环表示障碍物实体加机器人安全半径后的占用区域。

  • 目标距离 0.00 m
  • 安全航向 0.00 rad
  • 机器人安全半径 0.00 m
  • 控制状态 运行中

运行方式

仓库构建后，可以使用统一入口启动仿真，再通过控制台 TUI 切换模式。

source .vscode/project-terminal-init.sh
colcon build
source install/setup.bash

ros2 launch kibot_one_sim kibot_one.launch.py

# 另开一个终端
source .vscode/project-terminal-init.sh
source install/setup.bash
ros2 run kibot_one_control control_console
